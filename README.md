# English Learning App

Бұл Laravel арқылы жасалған web-қосымша.  
Қолданушы ағылшын сөздерін үйреніп, олардың прогрессін бақылай алады.

## About Project

Қолданушы:

- сөздерді көре алады
- сөздерді сақтай алады
- known / unknown деп белгілей алады
- деңгей бойынша прогрессті көре алады

## Tech Stack

- PHP 8
- Laravel
- MySQL
- Blade

## Installation

git clone https://github.com/username/project.git

cd project

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve

## Features

- Save words
- Track learning progress
- Level completion
- User profile statistics
